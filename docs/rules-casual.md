# かんたんルール説明

友達から部屋のURLをもらった人向けの、数分で読めるルール説明です。細かいことは抜きにして、まず遊ぶために必要なことだけ書いています。

このゲームは、サイコロ3個と3種類の「パワー」を使ってカードを場に設置していき、20VP（勝利点）を目指す拡大再生産系のボードゲームです。

## ゴール（勝ち方）

- 誰かが**20VP**に到達すると、そこで即終了ではなく、残りの全員があと1手番ずつプレイできます
- 全員が最後の手番を終えたら、そこでのVPを比較して一番多い人の勝ち
- 同点の場合は、設置したマーケットカードの枚数が多い方が勝ちです

## 1ターンの流れ

1. **ターン開始**：キャラクターや、すでに設置してあるカードの「ターン開始時効果」が発動することがあります
2. **サイコロを振る**：3個（キャラクターによっては4個）振ります
3. **好きな順番・好きな回数で行動**：以下を自由に組み合わせて行えます
   - カードを買う（手札に加える）
   - カードを設置する
   - 設置済みのカードを使う
   - レジェンドカードを置く（1ターンに1枚まで）
   - マーケットを入れ替える
   - パワーを別の種類に変換する
4. **ターン終了**：手札が上限（カード＋サイコロパン合わせて6枚）を超えていたら捨てて、次の人の番になります

## 基本用語

- **サイコロの出目とパワー**：出目1・2＝お金、3・4＝権力、5・6＝魔力になります
- **サイコロパン**：追加のサイコロのように使えるカードです。ただしカードを買うときや、出目そのものが必要なコストには使えません
- **カードの種類**：
  - キャラクター（1人1枚、8種類）
  - スタンダード（基本カード）
  - アドバンス（オープンマーケットに並ぶカード）
  - レジェンドⅠ・Ⅱ（高VPを狙える上級カード）
- **オープンマーケット**：場に並んでいる、みんなが買えるアドバンスカードのことです

コスト（サイコロ何個必要か、など）やカードごとの細かい効果はここには書いていません。気になるカードは右クリックすればいつでも確認できます（下記参照）。

## キャラクターについて

キャラクターは8種類いて、それぞれ得意なことが違います。ゲーム開始時に1人1キャラクターを選びます。

- **純白の女王 コロネリア**：サイコロが4個。毎ターン好きなパワーを1つもらえる
- **神眼の錬金術師 ソフィ**：毎ターンマーケットカードを2枚もらえる。手札を捨てていつでもパワーに変換できる
- **煌星の魔導将軍 アン**：サイコロパンをどんどん溜め込める
- **誓約の魔法騎士 フラン**：同じ名前のカードを手札に持っていれば、1ターンに1回タダで一緒に設置できる
- **浄罪の精霊術師 クロワ**：ゲーム中1回だけ、手番を丸ごと使ってマーケットカードをタダで設置できる。さらに毎ターンお金を1つもらえる
- **慈光の聖女 ショコラ**：毎ターン好きなパワーを2つもらえる。自分のパワーをいつでも別の種類と交換できる
- **傀儡の支配者 マリー**：カード設置に必要なサイコロが常に1個免除される
- **焔天の紋章術師 ロール**：開始時にパワーを合計10個好きな配分で獲得する（他のキャラは4個）。毎ターン権力パワーを1つもらえる

## 困ったら

カードを右クリックすると、コストや効果の詳細がポップアップで見られます。迷ったらまずそれを確認してください。
