# カードデータの確度について

このディレクトリのカードデータは、公式サイト・公式FAQ・コミュニティの攻略記事を突き合わせて再構成したものです。

- **キャラクター**（`characters.ts`）: 能力の概要はおおむね確認できているが、細かい数値（例：ロールの初期パワー配分数）は一部推定を含む。
- **スタンダードカード**（`standard.ts`）: 効果の種類（何を生産するか等）は確認できているが、**設置コストは未確認**（`cost: null`）。
- **アドバンスカード**（`advance.ts`）: 効果概要のみ確認できており、**コスト・VPともに未確認**。
- **レジェンドカード**（`legend.ts`）: 設置条件の概要のみ確認できており、**正確な数値・VPは未確認**。

`cost` または `victoryPoints` が `null` のカードは、`hasProgrammaticEffect: false` としてエンジン上は「設置不可（未実装）」として扱われる。実装を進める際は、物理カードまたは対象ゲームの公式ルールブックで数値を確認し、このデータを更新すること（このリポジトリでは対象ゲームの特定につながるURL・固有名詞は意図的に記載していない）。
